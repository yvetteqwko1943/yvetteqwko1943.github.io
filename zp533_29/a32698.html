SMALLRIG　Sony FX30 / FX3用カメラケージ　SR4183

【商品解説】
カメラを保護し、アクセサリーのマウントポイントを提供するように設計されています。
モジュラーケージは、NATOレールとU字型ケージで構成されています。
NATOレールは上部を1/4-20ネジ2本で固定し、取り外すことでオリジナルのXLRトップハンドルを取り付けることができます。
U字型ケージは両側と底面を1/4-20ネジで固定し、ねじれを防ぎ、傷に対するクッションを付けています。
バッテリードアには、カメラのカーブに合わせたヒンジアームがあり、小指での握り心地を向上させています。
HDMIケーブル専用のクランプでインターフェイスを保護し、安定した信号伝送を実現します。
ケージには、マジックアーム2070 など用の1/4-20穴が複数設けられています。
ARRI 3/8-16の位置決め穴は、トップハンドル2165C または モニターマウント2903B を接続します。
上部のNATOレールは トップハンドル3322 、左側レールは サイドハンドル2187B をサポートし、左上と左側面の1/4-20の位置決め穴は、フォローフォーカス用のロッドクランプ3598 を取り付けられます。
右側面の2つの1/4-20穴は、垂直撮影用にクイックリリースプレートにしっかりと取り付けることができます。
右レンズの中央には1/4-20の穴を2つ残し、クイックリリースプレートによる縦位置撮影時のたわみを防止しています。
コールドシューマウントを内蔵し、マイクやライトなどの取り付けが可能と、底部のQDポートにはQDストラップが装着可能です。
底部のアルカスイスクイックリリースプレートは、DJI RS 2 / RSC 2 / RS 3 / RS 3 Proを直接取り付けるか、DJI RS 2 / RSC 2ジンバルとアルカタイプクイックリリースプレート3154 用のクイックリリースプレート（高さ増加パッド）を取り付けることが可能です。
リストストラップ3848、ショルダーストラップPSC2428 などのアクセサリーに対応した穴あけ加工が両サイドと底面に施されています。
また、底面にマグネットスパナを内蔵し、簡単に分解・取り付けができ、すぐに使用できます。

●主な特長1. Sony FX30 / FX3を保護するフルケージ。
2. 取り外し可能なNATOレールを備えたモジュラーデザイン、オリジナルのXLRトップハンドルに対応。
3. バッテリードアのヒンジアームがカメラのカーブにフィットし、グリップ感が向上。
4. HDMIケーブル専用クランプにより、安定した信号伝送が可能。
5. 1/4-20ネジ穴、ARRI 3/8-16位置決め穴、コールドシューマウント、NATOレール、ハンドル、マイク、LEDライト、モニター、ハンドストラップなどのためのQDポート。
6. DJI RS 2 / RS 3スタビライザーとArca-Swiss三脚を素早く取り付けるためのアルカクイックリリースプレートを底面に装備。

●ご注意
1. オリジナルのXLRハンドルを取り付ける前に、上部のNATOレールを取り外してください。
2. 上部の1/4-20穴の中心距離は27mmですので、モニターマウント2904 のような中心距離が18mmのアクセサリーとは互換性がありません。

アシンメトリーに配置したスイッチボックスが独特のシルエット

アルミダイキャスト製のこだわり抜いたパーツのみで構成された ソケットユニットです。アシンメトリーに配置したスイッチボックスが独特のシルエットとインダストリアル感を生み出しています。
 
 
- Features -
Function 01
重厚感

アルミニウム合金のダイキャストで作られたソケットカバーやホルダーパーツなど、拘りぬいたパーツのみで構成されています。
   
Function 02
ギミック

インダストリアル感を助長するスイッチボックスにレバーを配置しON/OFF出来るギミックを搭載。
   
Function 03
佇まい

存在感あるソケットは点灯時だけでなく、消灯時にもその輝きを放っています。

※電球は付属していません。
   
Function 04
AMP PLUG SYSTEM

プラグ部分にはコードの長さを調整した際に余ったコードやワイヤーをスッキリと収納する工夫があります。またシーリングカバーは取り外しが可能で、お客様の好みによってお選びいただけます。※シーリングカバーの色はソケットと同系色となります。
 
Function 05
サイズ感

デザインと性能を両立させストレスのないサイズ感を追求。

- Brand information -
 
 
 

The ampersand (&) is a symbol that creates connections and means ‘and’.
Various connections exist between objects (products), events (lifestyle) and people (users and sellers, creators).
We take another look at these and create new value. This is why we chose the name of AMPERSAND FACTORY.



アンパサンド（ampersand、＆) とは「… と …」を示す「つながり」を生み出す記号。
モノ（製品 …）・事（ライフスタイル …）・人 ( 使い手や売り手、作り手 …）それらに存在する様々な「つながり」
それをもう一度見つめ直し、新たな価値を生み出す。
AMPERSAND FACTORY はそんな想いから名付けました。

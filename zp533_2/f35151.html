ITU-1770にも対応する、ピークメーター・プラグイン！

本製品は、ITU-1770 、IEC 268-10 Type I (Nordic N9)、IEC 268-10 Type IIa (BBC PPM)、IEC268-10 Type IIb (EBU PPM)、DIN 45406 などに対応した、ピークメーター・プラグインです。

ニードル、バー・タイプ、モノラル、ステレオ、サラウンドといった複数のメーター形式を選択可能。
そのほか、ニードルの動作について、ピークに達するまでの時間、ピークに達してから元に戻るまでの時間を設定可能、画面サイズ（縦横比固定）を自在にリサイズ可能など、かゆいところにも手が届く仕様。
様々なサウンドが渦巻く映像の世界を、PPmulator で制圧しましょう！

【起動方法】
スタンドアロン：サウンドカードから入力されたきた信号を、メーター上で表示します。
プラグイン起動（VST、AU、RTAS）：基本的にはDAW 上のマスターアウトプットに挿して使用します。

【メータータイプ】
IEC 268-10 TYPE I: Nordic N9
IEC 268-10 TYPE I: DIN 45406
IEC 268-10 TYPE IIa: BBC Scale
IEC 268-10 TYPE IIb: EBU Scale
ITU-R BS. 1770-2: "True-Peak" Level

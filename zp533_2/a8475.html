説明: Overview
PoE Ethernet / USB HUB HAT for Raspberry Pi Zero series boards, 3x USB 2.0 Ports, 802.3af-compliant
The PoE/ETH/USB HUB HAT is a PoE (Power over Ethernet) USB HUB HAT designed for Raspberry Pi Zero / Zero W / Zero WH. By using this small HAT and some proper 802.3af-compliant power sourcing equipments, it is possible to provide both network connection and power supply for your Raspberry Pi in only one Ethernet cable, along with 3x extended USB ports.
Features
Designed for Raspberry Pi Zero Series Board, compatible with Raspberry Pi Zero / Zero W / Zero WH
Incorporates RTL8152B Ethernet chip, with 10M / 100M auto-negotiation RJ45 port
PoE (Power over Ethernet) feature, 802.3af-compliant
Fully isolated SMPS (Switch Mode Power Supply)
3x extended USB 2.0 ports, compatible with USB 2.0 / 1.1

Specifications
POE power input: 37V ~ 57V DC in
Power Output: 5V 2.5A DC out
Network standard: IEEE 802.3af-compliant PoE
Dimensions: 23.10 × 65.00 mm
